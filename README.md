# spoo-me Scoop Bucket

Scoop manifests for [spoo.me](https://spoo.me) tools (Windows).

## Install

```powershell
scoop bucket add spoo-me https://github.com/spoo-me/scoop-bucket
scoop install spoo
```

Manifests in this bucket are published automatically by [GoReleaser](https://goreleaser.com) from [spoo-me/spoo-cli](https://github.com/spoo-me/spoo-cli) releases — they are not edited by hand.
